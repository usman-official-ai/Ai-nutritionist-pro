AI-Powered Personalized Diet Planning System

## Features

- 🤖 AI-Powered meal recommendations
- 📊 BMR and TDEE calculation
- 🍽️ Personalized meal plans
- 💪 Goal-based planning (Loss/Maintain/Gain)
- 🧴 Skin-specific dietary advice
- 📥 Download reports as TXT
- 📈 Macro tracking (Protein, Carbs, Fats)

## How It Works

1. Enter your details (Age, Gender, Weight, Height)
2. Select activity level (1-5)
3. Choose your goal (Weight Loss/Maintain/Weight Gain)
4. Get personalized diet plan instantly
